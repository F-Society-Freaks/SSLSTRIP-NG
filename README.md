# SSLSTRIP-NG
A simpe automated man in the middle attack tool powered by (nmap, sslstrip, arpspoof, urlsnarf, & driftnet).
#
# INSTALLATION

`$ git clone https://github.com/mkdirlove/SSLSTRIP-NG`

`$ cd SSLSTRIP-NG`

`$ chmod +x sslstrip-ng.sh`

`$ bash sslstrip-ng.sh`
